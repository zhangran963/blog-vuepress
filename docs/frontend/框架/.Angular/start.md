[Angular-Cli地址](https://github.com/angular/angular-cli/wiki/generate)



### 安装
* `npm install -g @angular/cli`安装 angular;

### 创建
* `ng g 名称`: 生成新项目;
* `ng g [类型] [名称]`: 生成[类型];
    * `class`
    * `component` 或 `c`: 组件(页面);
    * `directive`
    * `enum`
    * `guard`
    * `interface`
    * `module`
    * `pipe`
    * `service`
    * `appShell`
    * `application`
    * `library`
    * `universal`



### 启动
* `ng serve` 启动项目;
* `ng serve --open` 启动项目&打开浏览器;
