### 页面由哪三层构成

* 结构层
  + 由 HTML 或 XHTML 之类的标记语言创建; 
  + 描述内容的语义含义; 

* 表示层
  + 如何显示内容; 

* 行为层
  + 页面的内容如何对事件做出反应; 

### HTML5的优缺点

#### 优点
* 跨平台
* 成本底, 开发效率高, 快
* 容易推广
* 未来趋势, 发展中, 经常添加新特性; 
  + WebAssembly: 是一门低级的类汇编语言。它有一种紧凑的二进制格式，使其能够以接近原生性能的速度运行; 
  + WebRTC: 视频聊天，音频聊天; 
  + PWA: 渐进式Web应用; 是可被发现、易安装、可链接、独立于网络、渐进式、可重用、响应性和安全的; 

#### 缺点
* 受网络的限制
* 性能差, 所以用户体验差
