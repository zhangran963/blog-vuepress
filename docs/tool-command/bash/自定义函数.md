### 自定义函数
* `函数名 参数1 参数2 ...`: 调用函数
* 函数返回值在调用该函数后通过 `$?` 来获得
```sh
#!/bin/bash

# 定义函数
function getSum(){
  sum=$[$n1+$n2]
  return $sum
}

# 等待输入值
echo 计算两个数的和: 
read -p 第一个数: n1
read -p 第二个数: n2

# 执行函数并传参
getSum $n1 $n2
# 函数返回值在调用该函数后通过 $? 来获得
getSumRes=$?

echo ${n1}和${n2}的和是${getSumRes}
```
