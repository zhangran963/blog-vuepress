* transition: 过渡, 是一种简写属性;

按顺序显示:
* transition-property: 过渡属性的名称, 如 width, opacity;
* transition-duration: 过渡的时间, 如2s, 120ms;
* transition-timing-function: 动画的过程函数:
    * linear
    * ease
    * ease-in
    * ease-out
    * ease-in-out
    * step-start
    * step-end
    * steps(4, end)
    * cubic-bezier( x1, y1, x2, y2 )
    * ...
* transition-delay: 过渡开始的延时时间, 如2s, 4ms;
