## URI 统一资源标识符

* 统一资源定位符（URL）: 指定为非正式的网址; 
* 统一资源名称（URN）: 用于在特定的命名空间资源的标识，以补充网址; 

```md
                       权限                 路径
        ┌───────────────┴───────────────┐┌───┴────┐
  abc://username:password@example.com:123/path/data?key=value&key2=value2#fragid1
  └┬┘   └───────┬───────┘ └────┬────┘ └┬┘           └─────────┬─────────┘ └──┬──┘
  协议        用户信息         主机名    端口                  查询参数          片段
```
